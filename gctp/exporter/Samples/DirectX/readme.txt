DirectX X-File  サンプルコンバータ
----------------------------------------

このフォルダには、LightWave 3Dで作成したオブジェクトファイルもしくはシーンファイル
をDirectX のX-File形式に変換するコマンドラインのコンバータのソースコードが含まれて
います。プラグインバージョンのコンバータは、ディストームのHPで公開していますが、
ユーザーインターフェースを除いて変換のエンジン部分はこのソースコードを使用しています。

DirectX 8からサポートされたSkinedMeshに対応しており、LightWave 6.5でのボーンウェイト
情報をSkinedMeshに変換しています。

● LWO2DX

このプログラムは、LWOBもしくはLWO2オブジェクトファイルを読み込み標準出力に変換結果を
書き出します。

使い方

C:\> LWO2DX  APPLE.LWO > APPLE.X

オプションの説明

-b			バイナリー形式で出力します。
-Oc			頂点カラーを出力します。
-Of			フレームを出力しません。
-On			頂点法線ベクトルを出力しません。
-Ou			テクスチャUV座標値を出力しません。
-Om			マテリアルを出力しません。
-Ot			テンプレートを出力しません。
-Tr			テクスチャV座標値を反転して出力します。
-v7			DirectX 7と互換形式で出力します。
-s scale	座標値にスケールを掛けます。
-p name		フレームの前置詞を指定します。（デフォルトは"Frame"）


● LWS2DX

このプログラムは、LWSシーンファイルを読み込み標準出力に変換結果を書き出します。

使い方

C:\> LWS2DX  SPACE.LWS > SPACE.X

オプションの説明

-b			バイナリー形式で出力します。
-Oc			頂点カラーを出力します。
-Of			フレームを出力しません。
-On			頂点法線ベクトルを出力しません。
-Ou			テクスチャUV座標値を出力しません。
-Om			マテリアルを出力しません。
-Ot			テンプレートを出力しません。
-Tr			テクスチャV座標値を反転して出力します。
-Me			マテリアルをフレームの外側に名前付きで出力します。
-Ms			マテリアルを外側に出力し、オブジェクト間で同名のサーフェイスを共有します。
-v7			DirectX 7と互換形式で出力します。
-L			ライトのフレームとアニメーションを出力します。
-C			カメラのフレームとアニメーションを出力します。
-Fk			キーフレームのみを出力します。（デフォルト）
-Fs			指定したステップでサンプリングしたフレームを出力します。
-Fb			キーフレームとサンプリングしたフレームの両方を出力します。
-R			出力する必要の無いキーフレーム（静止したモーション）は出力しません。
-d content	コンテントディレクトリを指定します。
-f steps	モーションのキーを出力する間隔をフレームで指定します。
			frameに0を指定するとキーフレームのみが出力されます。
-p name		フレームの前置詞を指定します。（デフォルトは"Frame"）
-m nMax		１つの頂点に影響するSkinWeightの最大値を設定します。



仕様に関する詳しいマニュアルは、http://www.dstorm.co.jp/dslib/ で公開されている
「DirectX Export」のオンラインマニュアルを参照して下さい。
